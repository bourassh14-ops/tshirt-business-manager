# Inventory & Pricing Calculator - Multi-Platform App

## 🚀 Quick Start

### Web (Immediate)
```
start index.html
```

### Desktop (Electron)
1. Install Node.js: [nodejs.org](https://nodejs.org)
2. ```
cd tshirt-business-manager
npm install
npm start
```

### Mobile PWA (Install like Native App)
**Your PC IP:** `10.142.176.154`

1. PC: ```
cd tshirt-business-manager
python -m http.server 8000
```
2. **Phone browser:** http://10.142.176.154:8000
3. **Add to Home Screen** → 📱 App installed (offline/export works)

## 📱 Features
- Spreadsheet table w/ **inline editing**
- **Selling price DH per piece**
- Auto-calcs: weight (0.25kg/piece), logistics, costs, **profit**
- **Export CSV** button
- Offline PWA + responsive mobile
